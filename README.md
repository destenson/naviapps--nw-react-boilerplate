# nw-react-boilerplate

* React Redux Router
* Hot Loading
* Redux DevTools
* Webpack
* ESLint
* [Yarn](https://yarnpkg.com)

## Installation

```bash
git clone https://github.com/naviapps/nw-react-boilerplate my-app
```

## Start

```bash
npm start
```

## Build

```bash
npm run build
```
